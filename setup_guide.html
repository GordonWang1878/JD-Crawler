1 工具简介

这是一个京东商品价格批量监测工具，主要功能：

  • 从 Excel 文件批量读取京东商品链接
  • 自动爬取每个商品的原价和促销价（到手价 / 补贴价）
  • 支持京东账号登录，获取准确的登录后价格
  • 自动保存登录状态（Cookies），下次运行无需重复登录
  • 提供 Web 操作界面，通过浏览器即可操作，无需命令行知识
  • 爬取结果导出为 Excel 文件，方便后续分析

2 环境准备

在开始之前，请确保你的电脑上安装了以下软件：

2.1 Python 3.9+

检查是否已安装：

python3 --version

如果未安装或版本低于 3.9：

操作系统安装方式
macOS 方式一（推荐）：安装 Homebrew 后运行 brew install python
方式二：前往 python.org 下载安装包
Windows 前往 python.org 下载安装包
安装时务必勾选 "Add Python to PATH"

2.2 Google Chrome 浏览器

工具依赖 Chrome 浏览器来爬取数据。请确保已安装 Google Chrome（非 Chromium、非 Edge）。

💡 工具会自动检测你的 Chrome 版本并下载匹配的 ChromeDriver，无需手动配置。

2.3 Git（可选）

用于从 GitHub 下载代码。如果你不熟悉 Git，也可以直接下载 ZIP 压缩包（见下一步）。

检查是否已安装：

git --version

3 下载项目代码

方式一：使用 Git 克隆（推荐）

打开终端（macOS）或命令提示符（Windows），运行：

git clone https://github.com/GordonWang1878/JD-Crawler.git
cd JD-Crawler

方式二：下载 ZIP 压缩包

  1. 在浏览器中打开项目 GitHub 页面
  2. 点击绿色的 "Code" 按钮
  3. 选择 "Download ZIP"
  4. 解压到你想放置的目录
  5. 在终端中 cd 进入解压后的文件夹

4 安装 Python 依赖

在项目根目录下运行：

pip3 install -r requirements.txt
⚠️ Windows 用户注意：如果 pip3 命令不存在，请尝试使用 pip 代替。

安装完成后，可以验证关键包是否安装成功：

python3 -c "import flask; import selenium; import undetected_chromedriver; print('✓ 所有依赖安装成功')"

5 启动服务

在项目根目录下运行：

python3 app.py

看到以下输出说明启动成功：

======================================================================
JD Price Crawler
======================================================================

Open: http://localhost:5001

Press Ctrl+C to stop

 * Running on http://127.0.0.1:5001
✅ 现在打开浏览器，访问 http://localhost:5001 即可看到操作界面。
⚠️ Windows 用户注意：如果 python3 命令不存在，请尝试使用 python app.py。

停止服务：在终端中按 Ctrl + C 即可停止。

6 使用方法

6.1 上传 Excel 文件

  1. 打开浏览器，访问 http://localhost:5001
  2. 在页面中点击上传区域，选择包含京东商品链接的 Excel 文件（.xlsx 格式）
  3. 上传成功后，页面会显示识别到的商品数量

6.2 开始爬取

  1. 点击 "开始爬取" 按钮
  2. 工具会自动打开一个 Chrome 浏览器窗口（这是爬虫在工作，请不要关闭它）
  3. 首次扫码登录后，工具会自动 warmup（模拟正常浏览约 30 秒）再开始抓取
  4. 页面上会实时显示爬取进度和日志；请求间隔随批次逐渐增大（前 10 条 2-4 秒，之后 4-7 秒 / 6-10 秒）
  5. 若连续 2 条失败，会自动触发 30/60/90 秒冷却并重置会话

6.3 首次登录京东

💡 首次运行时，工具需要登录你的京东账号才能获取准确价格。
  1. 爬虫启动后会自动打开京东登录页面
  2. 在弹出的 Chrome 窗口中手动登录京东账号（推荐使用京东 App 扫码登录）
  3. 登录成功后，工具会自动检测到登录状态并继续爬取
  4. 登录信息（Cookies）会自动保存为 jd_cookies.pkl 文件
  5. 下次运行时会自动使用已保存的登录状态，无需重复登录
⚠️ 如果长时间未使用，Cookies 可能会过期，届时需要重新登录。工具会自动提示。

6.4 下载结果 / 重试失败项

  1. 爬取完成后，页面会显示统计信息（成功数、失败数、下架数、用时等）
  2. 点击 "下载结果" 按钮，即可下载 Excel 结果文件
  3. 结果文件也会保存在项目的 outputs/ 目录下
  4. 如果有失败或被拦截的条目，可以点击 "重试失败项"，工具会复用浏览器会话只重新抓取这些条目

7 Excel 文件格式说明

上传的 Excel 文件可以包含 5 列信息，工具会全部读入并在"数据预览"标签页显示：

字段可识别的列名（任一即可）说明
品牌 Brand / 品牌 仅展示，便于识别
型号（Item） Item / Model / 型号 仅展示，用于在结果中识别商品
商品链接 URL / ProductUrl Std / 链接 抓取目标。缺失时由 Product Key 自动构造
商品 ID Product Key / ProductKey / SKU 京东商品 ID（例：100140584252）
参考价 Price Reference / 参考价 仅展示，便于人工对比
💡 列名匹配规则：不区分大小写、空格、下划线，并支持前缀变体。例如 Price Reference_0、BRAND、ProductUrl Std 都能被识别。
✅ 最简要求：至少有 URL 或 Product Key 其中之一，其他字段可为空。

8 输出结果说明

爬取完成后会生成 Excel 文件（命名格式：Price_Marks_时间戳.xlsx），保存在 outputs/ 目录下，包含以下列：

列名说明
Batch Time本次批量爬取的开始时间
Crawl Time该商品的爬取时间
Brand品牌（来自输入）
Item型号（来自输入）
URL商品链接
Product Key京东商品 ID
Price Reference参考价（来自输入）
Status抓取状态，详见下表
Price原价（日常价 / 划线价）
Promotion Price促销价（到手价 / 补贴价）

Status 字段取值：

值含义可重试
success成功抓到原价和促销价—
partial只抓到其中一个价格✓
failed未知错误或无法提取价格✓
blocked反爬拦截（重定向到首页或验证页）✓
forbidden403 禁止访问✓
unavailable商品已下架（页面显示"已下柜"等）—
not_found商品不存在（URL 重定向到 www.jd.com/?d）—
💡 点击 Web 界面的 "重试失败项" 会自动重新抓取所有标记为可重试的条目。

9 常见问题排查

Q: 启动时提示端口 5001 被占用？

说明有其他程序占用了 5001 端口。解决方法：

# macOS / Linux：找到并终止占用端口的进程
lsof -ti:5001 | xargs kill -9

# 然后重新启动
python3 app.py

Q: 出现 ChromeDriver 版本不匹配的错误？

错误信息类似：This version of ChromeDriver only supports Chrome version XXX

通常是 ChromeDriver 缓存过期导致的。解决方法：

# 升级 undetected-chromedriver 包
pip3 install --upgrade undetected-chromedriver

# 重新启动
python3 app.py

Q: 爬取时大量商品显示 "blocked" 或 "forbidden"？

京东的反爬机制触发了。工具已内置冷却和请求节流，但若 IP 被系统级标记，仍会全批失败。建议：

  • 暂停 2-3 小时后再重试（严重时可能需要切换网络 / 更换 IP）
  • 确保已登录京东账号
  • 单次爬取控制在 200 条以内，避免短时间内反复批量跑
  • 用完后不要立即关闭浏览器窗口，工具会复用会话降低风险

Q: 首次启动时，"Initializing browser..." 卡几分钟？

undetected-chromedriver 需要从 Google 服务器下载匹配的 ChromeDriver。国内网络访问较慢时首次可能耗时 3-5 分钟。

  • 下载完成后会缓存到 ~/Library/Application Support/undetected_chromedriver/（macOS），后续启动只需数秒
  • 请耐心等待，不要手动终止进程——中断会导致下载失败，下次还要重新下载
  • 如果实在无法下载，可尝试开启代理后重新启动

Q: 为什么一条正常可访问的商品被标记为 blocked？

这是反爬保护：被重定向到首页时，除非 URL 明确是 https://www.jd.com/?d（下架/失效商品），否则都按 "可重试" 处理。点击"重试失败项"即可。

Q: Cookies 过期了怎么办？

删除项目根目录下的 jd_cookies.pkl 文件，然后重新运行。工具会引导你重新登录。

# 删除旧的 cookies
rm jd_cookies.pkl

# 重新启动
python3 app.py

Q: Windows 上运行 python3 提示找不到命令？

Windows 上 Python 命令通常是 python（不带 3），同理 pip 代替 pip3：

python app.py

Q: 安装依赖时报错 "pip not found"？

尝试以下命令：

# 方式一
python3 -m pip install -r requirements.txt

# 方式二（Windows）
python -m pip install -r requirements.txt
