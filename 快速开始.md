# 京东价格监测工具 - 快速开始

## 🚀 快速开始（3步搞定）

### 步骤1: 运行程序

```bash
python3 main_with_login.py
```

### 步骤2: 登录京东

程序会自动打开Chrome浏览器：
- 使用**手机京东APP扫码登录**（最简单）
- 或输入账号密码登录
- 登录成功后，回到终端按**Enter键**

### 步骤3: 选择模式

```
选择运行模式:
  1. 测试模式（只爬取前3个URL）  ← 首次运行推荐选这个
  2. 完整模式（爬取全部194个URL）
请输入 1 或 2:
```

**首次使用建议选择"1"（测试模式）**，确认能正常获取价格后，再运行完整模式。

## 💡 重要提示

### 关于登录

- **首次运行**: 需要手动登录一次
- **后续运行**: 自动使用保存的登录状态，无需重复登录
- **登录失效**: 如果提示cookies失效，重新登录即可

### 关于时间

- **测试模式** (3个URL): 约1-2分钟
- **完整模式** (194个URL): 约15-30分钟

每个商品之间有2-4秒延迟，避免被京东反爬虫系统封禁。

### 关于价格准确性

✅ 使用 `main_with_login.py`（登录版本）：
- 可以获取准确的京东实时价格
- 需要京东账号登录

❌ 使用 `main.py`（无登录版本）：
- 无法获取准确价格（京东需要登录才显示价格）
- 不推荐使用

## 🔧 常见问题

### Q: 如果cookies过期了怎么办？
A: 重新运行程序，它会提示你重新登录。

### Q: 可以在服务器上运行吗？
A: 首次登录需要图形界面。建议在本地登录一次，然后将生成的 `jd_cookies.pkl` 文件复制到服务器，后续可以使用无头模式运行。

### Q: 价格获取失败怎么办？
A:
1. 检查网络连接
2. 确认已成功登录
3. 部分商品可能暂时无法访问，属于正常现象

### Q: 可以多久运行一次？
A: 建议每天运行1-2次。频繁运行可能被京东限制访问。
